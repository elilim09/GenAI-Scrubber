# genai-scrubber

Chrome/Edge extension that scrubs personal data from text and images locally before they are uploaded. Built with Manifest V3, TypeScript, React, and crxjs.

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Testing

```bash
pnpm run test
pnpm run e2e
```
